Xcode坑之一Invalid argument

  • 正在搞代码，运行程序时突然发现程序不能再次运行了，一运行就提示Invalid argument

  • 然后翻墙各种查啊，试了好多方法都不行，重启，重置，我用git回退了几个版本都不行。要疯了

  • 最后终于找到问题所在了，原来是工程中的Bundle ID 不知道什么时候变为空了，重新设置一个就ok了。

  • 真是坑啊，花了几个小时就这个结果。
